# Taller 01: Stakeholders y Materialidad en el Sector Web

**Módulo:** Sostenibilidad Aplicada al Sistema Productivo
---

## Descripción

Análisis de *materialidad* ASG aplicado a **NexoWeb Consulting S.L.**, una consultora ficticia de desarrollo web. Se identifican los principales *stakeholders* y los temas materiales más relevantes bajo los tres pilares: **Ambiental**, **Social** y **Gobernanza**.

---

## Archivos

- `materialidad.xml` — Documento principal con stakeholders y temas materiales ASG.
- `materialidad.xsd` — Esquema de validación de la estructura XML.

---

## Stakeholders identificados

| ID | Nombre | Tipo | Importancia |
|----|--------|------|-------------|
| STK-01 | Clientes empresariales | Externo | Alta |
| STK-02 | Equipo de desarrollo | Interno | Alta |
| STK-03 | Proveedores cloud | Externo | Media |

---

## Temas materiales ASG

| ID | Tema | Pilar ASG | Nivel |
|----|------|-----------|-------|
| TM-01 | Eficiencia energética del software | Ambiental | Crítico |
| TM-02 | Accesibilidad web universal | Social | Crítico |
| TM-03 | Privacidad de datos y cumplimiento RGPD | Gobernanza | Crítico |
